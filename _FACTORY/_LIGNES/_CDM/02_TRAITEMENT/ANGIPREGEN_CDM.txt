# ANGIPREGEN_CDM - STANDARD PIPELINE-READY

## Nouveaux champs structurels

- ANGLE_ID=
- POOL=
- FORBIDDEN_POOLS=
- STATUS=
- RESERVED=
- QUOTA=

## Règles

- Un angle réservé ne peut être réutilisé dans un autre pool
- Les angles détailés par angles réservés doivent être explicites
- StatUS valides : ACTIVE / RESERVED / FORBIDDEN

## Exemple normalisé

ANGLE_ID=JOU-03-A2
POOL=QV-01
FORBIDDEN_POOLS=IF-ROT-02
STATUS=ACTIVE

ANGLE_ID=JOU-19-A3
POOL=QVM-07
STATUS=RESERVED
RESERVED=CDM-BUE-09

## Objectif

- supprimer l'interprétation humaine implicite
- stabiliser la génération
- préparer l'automatisation FACTORY
