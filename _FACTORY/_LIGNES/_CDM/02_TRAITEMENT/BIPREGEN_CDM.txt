# HISTORIQUE DES INTERVENTIONS

## État initial du document

Le fichier `BIB_CDM.txt` était une base de connaissances brute sur la Coupe du Monde, organisée en 9 sections thématiques. Chaque item portait un indicateur de difficulté en fin de ligne (F / I / D). Certains items s'étendaient sur plusieurs lignes. Aucun système de codage, aucune statistique, aucune légende.

## Actions réalisées

1. Mise en ligne unique — Sections 3 (Matchs) et 4 (Buteurs)
2. Tableau STATISTIQUES — Comptage des items par section et par niveau
3. Proportions — Colonnes "% réel" et "% cible (règles FACTORY)"
4. Ligne "Questions à extraire" — Objectif 300 questions (120/120/60)
5. Reclassement de 4 items N1 → N2 : CDM-JOU-18 (Neuer), CDM-JOU-20 (Modrić), CDM-MAT-12 (Finale 2010), CDM-FIN-10 (Angleterre)
6. Codage de chaque item — `CDM-[CAT]-[N°]-[NIV]`
7. LÉGENDE ajoutée
8. INDEX PAR NIVEAU ajouté avec quotas

# LÉGENDE

## Format du code
`CDM-[CAT]-[N°]-[NIV]`

| Champ | Détail |
|-------|--------|
| `CDM` | Quiz Coupe du Monde |
| `CAT` | Catégorie |
| `N°`  | Numéro d'item dans la catégorie (01, 02...) |
| `NIV` | Niveau de difficulté (N1 / N2 / N3) |

## Catégories

| Code  | Section |
|-------|---------|
| `JOU` | Joueurs |
| `STA` | Stades |
| `MAT` | Matchs |
| `BUT` | Buteurs toutes éditions |
| `BUE` | Meilleurs buteurs par édition |
| `FIN` | Finalistes |
| `PAR` | Participations |
| `ANE` | Anecdotes / Polémiques |
| `REC` | Records |

## Niveaux

| Code | Signification |
|------|---------------|
| `N1` | Niveau 1 — Facile |
| `N2` | Niveau 2 — Intermédiaire |
| `N3` | Niveau 3 — Difficile |
